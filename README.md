## Lab 01 - System Health Monitor

This lab includes a Bash script that collects system health information such as:

* System information
* Kernel version
* Uptime
* Logged-in users
* Memory usage
* Disk usage
* Log directory size
* Top CPU-consuming processes
* Top memory-consuming processes
* Network interfaces
* Listening ports
* Recent system errors
