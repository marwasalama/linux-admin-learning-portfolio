# Linux System Health Monitor

A simple Linux system administration project that uses a Bash script to collect system health information and generate a monitoring report.

## Project Objective

The purpose of this project is to practice Linux system monitoring tasks commonly used by Linux System Administrators and Infrastructure Engineers.

The script collects information about:

* System information
* Kernel version
* System uptime
* Logged-in users
* Memory usage
* Disk usage
* Log directory size
* Top CPU-consuming processes
* Top memory-consuming processes
* Network interfaces
* Listening ports
* Recent system errors

## How to Run

Give the script execute permission:

```bash
chmod +x health_check.sh
```

Run the script:

```bash
./health_check.sh
```

The script generates:

```text
system_health_report.txt
```
